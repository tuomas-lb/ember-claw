<task type="auto">
  <name>Task 2: Implement deploy, list, delete, status, and logs subcommands</name>
  <files>internal/cli/deploy.go, internal/cli/list.go, internal/cli/delete.go, internal/cli/status.go, internal/cli/logs.go</files>
  <action>
1. Create `internal/cli/deploy.go`:
  - func newDeployCommand() *cobra.Command
  - Args: cobra.ExactArgs(1) for instance name
  - Flags: --provider (required, string), --api-key (required, string), --model (required, string), --image (default "reg.r.lastbot.com/ember-claw-sidecar:latest"), --cpu-request (default "100m"), --cpu-limit (default "500m"), --memory-request (default "128Mi"), --memory-limit (default "512Mi"), --storage-size (default "1Gi"), --storage-class (default ""), --env (string-to-string map flag for custom env vars)
  - RunE: build DeployOptions from flags + arg, call k8sClient.DeployInstance(ctx, opts)
  - On success: print "Instance {name} deployed successfully" in green (use fatih/color)
  - Validate instance name against K8s regex `^[a-z0-9][a-z0-9-]{0,61}[a-z0-9]$` before calling deploy

2. Create `internal/cli/list.go`:
  - func newListCommand() *cobra.Command
  - No args
  - RunE: call k8sClient.ListInstances(ctx), format as table using tablewriter v1.1.3
  - Table columns: NAME, STATUS, READY, AGE
  - Use tablewriter v1 API: table.Header(headers), table.Bulk(rows), table.Render()
  - Age formatted as human-readable (e.g., "2h30m", "5d")
  - If no instances: print "No PicoClaw instances found"

3. Create `internal/cli/delete.go`:
  - func newDeleteCommand() *cobra.Command
  - Args: cobra.ExactArgs(1)
  - Flags: --purge (bool, default false)
  - RunE: call k8sClient.DeleteInstance(ctx, name). If --purge: warn about data loss, prompt "Confirm? [y/N]:", read stdin, if "y" call k8sClient.DeletePVC(ctx, name). Without --purge: print note about retained PVC.
  - Use fatih/color for warning text (yellow for PVC warning, red for data loss)

4. Create `internal/cli/status.go`:
  - func newStatusCommand() *cobra.Command
  - Args: cobra.ExactArgs(1)
  - RunE: call k8sClient.GetInstanceStatus(ctx, name)
  - Display as key-value pairs: Name, Status, Ready, Pod, Provider, Model, Age, CPU Limit, Memory Limit
  - Use tablewriter or simple fmt.Printf with aligned output

5. Create `internal/cli/logs.go`:
  - func newLogsCommand() *cobra.Command
  - Args: cobra.ExactArgs(1)
  - Flags: --follow/-f (bool), --tail (int64, default 100)
  - RunE: call k8sClient.GetInstanceLogs(ctx, name, follow, tail), io.Copy to os.Stdout
  - Handle ctx cancellation for clean Ctrl+C exit

6. Wire all commands in root.go: root.AddCommand(newDeployCommand(), newListCommand(), newDeleteCommand(), newStatusCommand(), newLogsCommand())
