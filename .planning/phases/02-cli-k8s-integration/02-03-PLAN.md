2. Implement runChat per Research Code Example "Full Port-Forward + gRPC Chat Flow":
  Step 1: k8sClient.FindRunningPod(ctx, instanceName) -- get pod name
  Step 2: k8sClient.PortForwardPod(ctx, podName, 50051) -- establish tunnel
  Step 3: grpcclient.DialSidecar(ctx, pf.LocalPort) -- connect gRPC
  Step 4a (single-shot, -m flag): svcClient.Query(ctx, &QueryRequest{Message: message, SessionKey: instanceName}), print resp.Text, check resp.Error
  Step 4b (interactive, no -m flag): readline loop per Research Pattern 7:
    - readline.New(fmt.Sprintf("[%s]> ", instanceName))
    - Open bidi stream: svcClient.Chat(ctx)
    - Loop: rl.Readline() -> stream.Send(ChatRequest) -> stream.Recv() -> print resp.Text
    - Break on readline.ErrInterrupt or io.EOF
    - stream.CloseSend() on exit
  - defer close(pf.StopChan) to clean up port-forward
  - defer grpcConn.Close() to clean up gRPC connection
  - Print "Connecting to {instanceName}..." before port-forward (user feedback)
  - Print "Connected. Type messages or Ctrl+C to exit." before readline loop

3. Wire chat command in root.go: add newChatCommand() to root.AddCommand(...)
